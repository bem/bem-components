# image

`<img>`-based block for images, using as a content (logo, for example).

```bemjson
    {
        block : 'image',
        url : 'http://bem.info/m/_/20zuV2gLiyoaqrHv9LoBRTmB0Ow.svg',
        width : 16,
        height : 16,
        alt : 'BEM',
        title : 'Find us on bem.info'
    }
```
