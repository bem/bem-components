# image

Блок **image** используется для вставки картинок.

В HTML представлен тегом `<img>`.

```bemjson
    {
        block : 'image',
        url : 'http://bem.info/m/_/20zuV2gLiyoaqrHv9LoBRTmB0Ow.svg',
        width : 16,
        height : 16,
        alt : 'BEM',
        title : 'Find us on bem.info'
    }
```
