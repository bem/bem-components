# radio  

Блок `radio` – «радио-переключатель» используется для создания одиночных радиобоксов. 

В реализации блока используется функциональность нативного контрола радиобокс, который при этом скрыт. 

Блок `radio` в результате BEMHTML-преобразований на странице заменяется HTML-тегом `<label>` – контейнером, содержащим:

* элемент `radio__box`, содержащий скрытый контрол `input` (элемент `__control`);
* подпись к радио-кнопке, если задан BEMJSON-параметр `text`.

## Модификаторы

### Темы блока `_theme`

Блок представлен в следующих темах:

 * simple   
 * normal  

Наглядно видно на примерах ниже:


#### simple

```bemjson
{
    block : 'radio',
    mods : { theme : 'simple' },
    name : 'radio-simple1',
    text : 'Lonely switch1'
}
```

 
#### normal

```bemjson
{
    block : 'radio',
    mods : { theme : 'normal', size : 'l' },
    name : 'radio-normal1',
    text : 'Lonely switch2'
}
```

Без указания темы к блоку применяются значения по умолчанию (*default*), установленные браузером.

### Тип радио-переключателя `_type`

Модификатор `_type` со значением `button` используется для создания кнопочного радио-переключателя – кнопки (блок `button`) с установленным модификатором `_togglable_radio`. 

При этом к блоку `radio` примешиваются:

* блок `button`;
* элемент `button__controll`. 

Кроме того, при установленном модификаторе `_type_button` для блока задается HTML-атрибут `role="button"`. 

Модификатор реализован во всех темах блока.

### Неактивен `_disabled`
   
В состоянии "неактивен" радио-переключатель отображается, но недоступна для действий пользователя.  

Такой радио-переключатель не будет получать фокус (модификатор `_focused`).

При установке модификатора `_disabled` для радио-переключателя не будут:

* устанавливаться модификаторы состояния `_hovered`, `_pressed` и `_focused`;
* изменяться значение модификатора `_checked`.

```bemjson
{
    block : 'radio',
    mods : { 
        theme : 'normal', 
        size : 'l', 
        disabled : true 
    },
    name : 'radio-normal1',
    text : 'Lonely switch1'
}
```

Модификатор реализован во всех темах блока.



### Состояния блока 

#### В фокусе `_focused`

Модификатор `_focused` в значении `true` автоматически выставляется блоку в момент, когда он находится в фокусе. Например, по нажатию клавиши `Tab` или при щелчке мышью.

Реализован во всех темах блока.

```bemjson
{
    block : 'radio',
    mods : { 
        theme : 'normal', 
        size : 'l', 
        focused : true 
    },
    name : 'radio-normal1',
    text : 'Lonely switch1'
}
```

#### Выбран `_checked`

Модификатор `_checked` указывает на то, что радио-переключатель выбран. Модификатор устанавливается по:

* щелчку мыши по блоку (при повторных щелчках модификатор сохраняется);
* нажатию клавиш `Enter` или `Space`, если радио-переключатель находится в фокусе (модификатор `_focused`);
* нажатию клавиш стрелок, если фокус установлен на соседний радио-переключатель в радио-группе или в блоке, содержащем несколько блоков `radio`.

Реализован во всех темах блока.


```bemjson
{
    block : 'radio',
    mods : { 
        theme : 'normal', 
        size : 'l', 
        checked : true 
    },
    name : 'radio-normal1',
    text : 'Lonely switch1'
}
```


#### Состояния доступные с `_type_button`

При установленном модификаторе `_type_button` к блоку примешивается блок `button`. Как следствие, для блока с модификатором `_type_button` доступны состояния блока `button`:

*  `_hovered` – под курсором;
*  `_pressed` – кнопка нажата.


## Элементы

### `__box`

Элемент `__box` используется в стандартной реализации блока как контейнер, содержащий контрол радиобатона (элемент `__control`). В результате BEMHTML-преобразований становится HTML-элементом `<span>`.

### `__control`

Элемент `__control` – контрол радиобатона. В результате BEMHTML-преобразований становится HTML-элементом `<input>` с атрибутом `type="radio"`. По умолчанию элемент скрыт. В блоке используется только его функциональность.
